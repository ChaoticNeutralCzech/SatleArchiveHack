Satle Archive Hack

This guide allows you to add the option to play previous/next Satle games, effectively enabling an archive so you can binge-play.

How-to

  1. Install the Redirector addon for your browser.
  2. Right-click to save this file: SatleArchiveScript_Redirector.json
  3. Click the extension icon, Edit redirects and import the file. This will not affect your existing redirects.
  4. Reload Satle and you should see a number at the top. Pressing the arrows or typing another number should load another game.
